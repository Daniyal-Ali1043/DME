## Quickstart

1. Copy `.env.example` to `.env.local` and fill values.
2. `npm i`
3. Run migrations: `npm run drizzle:generate` then `npm run drizzle:push`
4. `npm run dev`

Deploy on Vercel. Configure cron in `vercel.json` and set required env vars.
